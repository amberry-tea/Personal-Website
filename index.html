Portfolio

Dimensional Chess (Jan 2020 - Present)

Academic / Personal Project

A chess program that supports any number of dimensions (greater than 1). The program renders a 2D cross-section of the board you are playing on and you can move along the 3rd or 4th dimension using the arrow keys.

  • Tasked with making chess for Java course, went above and beyond requirements.
  • User can choose the dimensions and size of the board.
  • Uses matrix algebra to deduce correct moves the player can make.
  • User can choose starting positions for pieces using the GUI and saving the layout as a file (in progress).

Equal Load Distribution Application (Jan 2020)

QDS Hackathon 2020

Software that improves the efficiency of Teck's mining operations by equally distributing trucks using traffic simulation and other data analysis techniques.

  • Collaborated in a team of 3: two first year computing students and one business student.
  • Used data analysis to calculate and visualize standard deviation using histograms.
  • Calculated the equal distribution of trucks to shovels for optimal operations.
  • Simulated site traffic to determine when trucks take left turns and how they can avoid other trucks.
